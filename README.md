# Space Explorer

An interactive browser-based space exploration game page built with Three.js. You start near Earth in our solar system, pilot an India-inspired spaceship in third-person view, and travel freely across a galaxy filled with stars, nebulae, solar systems, distant galaxies, and constellations.

## Play Online

Click here to Explore:
[https://amansharmadev.github.io/space-explorer/](https://amansharmadev.github.io/space-explorer/)

## Highlights

- Third-person spaceship exploration
- India-inspired spaceship styling
- Journey begins near Earth in our solar system
- Procedural planets with atmospheres and cloud layers
- Multiple solar systems and extra distant galaxy clusters
- Constellations visible in deep space
- Mobile-friendly touch controls
- Lightweight static setup with no build step

## Controls

### Desktop

- `W / S` or `↑ / ↓`: move forward or brake
- `A / D` or `← / →`: turn left or right
- `Q / E`: roll ship
- `R / F`: move up or down
- `C`: toggle camera mode

### Mobile

- Left touch pad: thrust and turning
- Right buttons: roll, rise, dive, and camera toggle

## Notes

- The scene is optimized to run better on mobile and lower-power devices.
- If the browser shows performance warnings, reload once and let the scene settle.
- The project is fully static and suitable for GitHub Pages hosting.
